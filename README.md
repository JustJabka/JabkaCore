# JabkaCore
JabkaCore adds a workbench in a barrel, with which you can add your own crafts with NBT tags to your datapack, useful predicates and tags.

## Description of JabkaCore
JabkaCore is core for datapacks in Minecraft. There are some features of JabkaCore : 
- It supports custom workbench, so you need to create only recipes.
- It has many useful predicates
- It has many useful tags
