# JabkaCore
JabkaCore adds a workbench in a barrel, with which you can add your own crafts with NBT tags to your datapack, useful scoreboards, block tags.

## Description of JabkaCore
JabkaCore is core for datapacks in Minecraft. There are some features of JabkaCore : it supports custom workbench, so you need to create only recipes.
